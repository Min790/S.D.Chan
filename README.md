# S.D.Chan (SOOP(Chzzk) Downloader Chan)

**S.D.Chan**은 원작자(스트리머)의 허락을 받은 영상 편집자가 작업용 소스 영상을 빠르고 편리하게 확보할 수 있도록 돕기 위해 개발된 고성능 다운로드 도구입니다.

---

## ✨ 주요 기능
- **초고속 FAVD 엔진**: 파일을 수십 개로 강제 분할하여 다운로드하는 자체 엔진을 탑재하여 압도적인 속도를 제공합니다.
- **SOOP & 치지직 & 씨미 지원**: 세 플랫폼의 모든 링크 형식을 지원합니다. (다시보기, 클립, 캐치)
- **사용자 친화적 GUI**: 복잡한 설정 없이 URL 입력과 화질 선택만으로 즉시 다운로드가 가능합니다.

## 🚀 사용 방법
1. `SDChan.exe` 파일을 실행합니다.
2. 다운로드하고자 하는 영상의 URL을 입력창에 붙여넣습니다.
3. 원하는 화질과 가속 모드(스레드 수)를 선택합니다.
4. **[다운로드 시작]** 버튼을 누릅니다.
5. 완료될 때까지 기다리면 `downloads` 폴더에 영상이 저장됩니다.
